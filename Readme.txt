Requirements for Athena: AI Addaptive Assistant

1.  System Requirements:

-   Windows 10/11
-   Working microphone
-   Internet connection
-   Python 3.8 or higher (PyCharm recommended)

2.  Required Python Libraries:

-   SpeechRecognition
-   pyttsx3
-   openai
-   pyaudio
